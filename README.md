## 程式遊戲規則
跳棋的棋盤有6個角，可供3人進行遊戲，每個人把各自同顏色的棋子擺滿一個角，按照規則輪流走棋，以最早旗子全部抵達並擺滿對角為優勝。對局將棋子互相間隔一個角擺放以平均分布。
每方棋子6枚，擺成等邊三角形。
## 程式內容
1. **Hex 結構體** - 六角座標系統的核心
2. **Board 類別** - 遊戲邏輯的主要管理者  
3. **std::hash<Hex> 模板特化** - 讓Hex能與STL容器配合
4. **封裝性**：Board類別的內部實作細節都是私有的
